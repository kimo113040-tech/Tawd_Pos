# 🛒 TAWD POS - نظام الكاشير المتكامل

نظام كاشير احترافي متكامل للمحلات والمطاعم، مصمم بواسطة **Karim Elgioushy** - شركة طود TAWD

## ✨ المميزات الرئيسية

### 💼 المبيعات
- **نقطة البيع (POS)** - كاشير رئيسي
- **تيك أواي (Take Away)** - طلبات الاستلام
- **دليفري (Delivery)** - طلبات التوصيل

### 📦 المخزون
- إدارة الأصناف مع الصور والباركود
- إدارة المخازن المتعددة
- جرد المخزن التلقائي
- تنبيهات المخزون المنخفض

### 👥 الأطراف
- إدارة العملاء مع نقاط المكافآت
- إدارة السائقين (اختياري)
- إدارة الموردين
- المديونيات والتحصيلات

### 💰 المالية
- المصروفات (إيجار، كهرباء، رواتب...)
- الفواتير والمرتجعات
- تقارير المبيعات الشاملة

### ⚙️ الإدارة
- الشيفتات (بدء/تقفيل)
- الحضور والانصراف
- إدارة المستخدمين
- جرد الخزينة

### 🔐 المصمم
- تفعيل البرنامج بأكواد فريدة
- إدارة أكواد التفعيل
- تقارير الأكواد

## 🚀 التثبيت والنشر

### المتطلبات
- متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- خادم ويب (اختياري للتشغيل المحلي)

### التثبيت المحلي
1. انسخ جميع الملفات إلى مجلد واحد
2. افتح `index.html` في المتصفح
